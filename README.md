# Mighty Burner Track Club Website

Static site (HTML/CSS/Vanilla JS) for Netlify + GitHub.

## Run locally (easy)
1. Open folder in VS Code
2. Install VS Code extension: **Live Server**
3. Right-click `index.html` -> **Open with Live Server**

## Deploy on Netlify
- Publish directory: `.`
- No build command needed

## Edit content
- Main page: `index.html`
- Styles: `assets/css/styles.css`
- JS: `assets/js/main.js`

## Netlify form
Registration form uses Netlify Forms automatically:
- Deploy once to Netlify
- Submit a test entry
- Then Netlify will show the form in the dashboard
